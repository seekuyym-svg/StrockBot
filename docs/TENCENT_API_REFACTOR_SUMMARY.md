# 腾讯财经API统一数据源重构总结

## 📋 重构概述

本次重构将股票行情数据获取统一使用**腾讯财经API**，移除了对雪球、AKShare、BaoStock等其他数据源的依赖。

## 🎯 重构目标

1. **简化数据源**：统一使用单一稳定的数据源（腾讯财经）
2. **减少依赖**：移除akshare、baostock等第三方库依赖
3. **提高稳定性**：腾讯财经API响应快速、数据结构稳定
4. **完整功能**：通过K线数据计算日/周/月涨跌幅

### 2. 数据获取流程

#### 日涨跌幅
- **来源**: 腾讯财经实时行情API (`http://qt.gtimg.cn/q={market}{code}`)
- **字段**: `parts[32]` (涨跌幅百分比)
- **优势**: 直接获取，无需计算

#### 周/月涨跌幅
- **来源**: 腾讯财经历史K线API (`http://web.ifzq.gtimg.cn/appstock/app/fqkline/get`)
- **计算方法**:
  - 周涨幅: 对比5个交易日前的价格 (索引 `-6`)
  - 月涨幅: 对比20个交易日前的价格 (索引 `-21`)
- **复权方式**: 前复权 (qfq参数)

#### 流通市值
- **来源**: 腾讯财经实时行情API
- **字段**: 
  - `parts[72]` 或 `parts[76]`: 流通股本（股数）
  - `parts[3]`: 当前价格
- **计算**: `流通市值 = 流通股本 × 当前价格 / 1e8` (转换为亿元)

## 📊 API接口详情

### 实时行情API
```
URL: http://qt.gtimg.cn/q=sz002706
编码: GBK
返回格式: v_sz002706="1~良信股份~002706~11.21~10.19~..."
```

**关键字段映射**:
| 索引 | 字段说明 | 示例值 |
|------|---------|--------|
| 1 | 股票名称 | 良信股份 |
| 3 | 当前价格 | 11.21 |
| 4 | 昨收价 | 10.19 |
| 32 | 涨跌幅% | 10.01 |
| 72/76 | 流通股本(股) | 914381190 |

### 历史K线API
```
URL: http://web.ifzq.gtimg.cn/appstock/app/fqkline/get
参数: param={market}{code},day,,,{days},qfq
返回: JSON格式，包含qfqday数组
```

**K线数据格式**:
```json
["2024-01-01", "开盘", "收盘", "最高", "最低", "成交量", "成交额"]
```

**注意**: 
- 部分数据可能只有6个字段（无成交额）
- 分红日数据第7个字段可能是字典，需跳过


### 3. 边界检查
```python
# 计算周涨幅需要至少6条数据
if len(klines) >= 6:
```

## 📌 注意事项

1. 腾讯财经API可能需要设置合理的请求间隔，避免被限流

3. K线数据中的分红日记录需要特殊处理
4. 流通市值是通过计算得出，非直接字段
