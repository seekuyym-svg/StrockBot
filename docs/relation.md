个股与指数之间的相关性，通常指的是**收益率序列的皮尔逊相关系数**。直接使用价格序列计算相关性容易出现“伪相关”（非平稳序列导致的高相关性），因此专业做法是基于**收益率**（或对数收益率）来计算。

如果你已经有历史K线数据（包含开盘、最高、最低、收盘、成交量等），按以下步骤即可计算个股与上证指数的相关性。

---

## 一、核心概念：皮尔逊相关系数

皮尔逊相关系数 \( r \) 衡量两个变量之间的线性相关程度，取值范围 \([-1, 1]\)：

- \( r > 0 \)：正相关（个股与指数同涨同跌）
- \( r < 0 \)：负相关（个股与指数走势相反）
- \( r = 0 \)：无线性相关性

计算公式为：

\[
r = \frac{\sum_{i=1}^{n}(X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum_{i=1}^{n}(X_i - \bar{X})^2} \cdot \sqrt{\sum_{i=1}^{n}(Y_i - \bar{Y})^2}}
\]

其中 \( X_i \) 和 \( Y_i \) 分别是个股和指数的**收益率序列**，\( \bar{X} \)、\( \bar{Y} \) 是各自收益率的均值。

---

## 二、具体计算步骤（使用历史K线数据）

假设你有一个包含个股和上证指数的日K线数据，日期对齐。

### 第1步：计算收益率序列

推荐使用**对数收益率**，因为它在时间上具有可加性，且更接近正态分布。

\[
\text{Return}_t = \ln\left(\frac{\text{Close}_t}{\text{Close}_{t-1}}\right)
\]

其中 \( \text{Close}_t \) 为第 \( t \) 天的收盘价。

也可以使用简单收益率（百分比涨跌幅）：
\[
\text{Return}_t = \frac{\text{Close}_t - \text{Close}_{t-1}}{\text{Close}_{t-1}}
\]

两种方法计算出的相关性数值非常接近，实际分析中均可。

**注意事项**：
- 需要剔除停牌日或无交易日的缺失数据
- 个股和指数的日期必须严格对齐（同一天）

### 第2步：选择时间窗口

相关性会随时间变化，因此需要明确计算区间：

- **长期相关性**：使用全部历史数据（如上市以来）
- **滚动相关性**：固定窗口（如60天、120天、250天）每天计算一次，观察相关性变化趋势

常见窗口：
- 60天（约3个月）：短期相关性
- 120天（约半年）：中期相关性
- 250天（约1年）：长期相关性

### 第3步：计算皮尔逊相关系数

在Excel中可用 `=CORREL(个股收益率范围, 指数收益率范围)`

在Python中（pandas）：
```python
import pandas as pd
import numpy as np

# 假设df包含日期、个股收盘价、指数收盘价
df['ret_stock'] = np.log(df['close_stock'] / df['close_stock'].shift(1))
df['ret_index'] = np.log(df['close_index'] / df['close_index'].shift(1))
df = df.dropna()  # 删除缺失值

# 计算整个区间的相关性
corr_full = df['ret_stock'].corr(df['ret_index'])

# 计算滚动相关性（例如60天窗口）
df['corr_60d'] = df['ret_stock'].rolling(60).corr(df['ret_index'])
```

### 第4步：解读结果

| 相关系数范围 | 解释 |
|-------------|------|
| 0.7 ~ 1.0 | 高度正相关（强β股，如银行、券商等权重股） |
| 0.3 ~ 0.7 | 中度正相关（大部分个股） |
| -0.3 ~ 0.3 | 弱相关或无关（如防御性个股或独立题材股） |
| -0.7 ~ -0.3 | 中度负相关（较少见，如黄金股与大盘有时负相关） |
| -1.0 ~ -0.7 | 高度负相关（罕见） |

## 三、常见问题与注意事项

### 1. 为什么不直接用价格序列计算相关性？

价格序列是非平稳的（存在趋势和漂移），即使两个毫无关系的序列也可能因同时上涨而得出高相关性。**收益率序列**消除了趋势，反映的是每日波动之间的同步性，更符合“相关性”的金融含义。

### 2. 应该使用日收益率、周收益率还是月收益率？

- **日收益率**：最常用，灵敏度高，但受微观噪声影响。
- **周收益率**：可过滤部分日内噪声，更适合中长线分析。
- **月收益率**：平滑度最高，但样本点较少。

对于短期交易策略分析，日收益率足够。

### 3. 如何处理停牌或非同步交易日？

- 将个股和指数的数据按日期对齐后，如果某一天个股停牌（无价格），直接删除该行（因为指数仍在交易，无法配对）。
- 也可以使用前复权价格，避免除权除息导致的收益率异常。

### 4. 相关性与贝塔（β）的区别

- **相关系数** \( r \)：衡量两者线性关系的紧密程度，无量纲。
- **贝塔系数** \( \beta \)：衡量个股相对于指数的波动敏感度，\( \beta = r \cdot \frac{\sigma_{\text{stock}}}{\sigma_{\text{index}}} \)。

例如：个股与指数相关系数为0.8，个股波动率是指数的1.5倍，则 \( \beta = 0.8 \times 1.5 = 1.2 \)，表示指数涨1%，个股平均涨1.2%。

## 四、实战案例（Excel快速计算）

假设你有以下数据（A列日期，B列个股收盘价，C列上证指数收盘价）：

| 日期 | 个股收盘 | 上证收盘 |
|------|---------|---------|
| 2025-04-14 | 10.00 | 3000 |
| 2025-04-15 | 10.20 | 3050 |
| 2025-04-16 | 10.10 | 3040 |
| ... | ... | ... |

**步骤**：
1. D列计算个股日收益率：`=LN(B3/B2)` 向下填充
2. E列计算指数日收益率：`=LN(C3/C2)` 向下填充
3. 使用公式 `=CORREL(D2:D100, E2:E100)` 得到相关系数

> 💡 如果只想看近期（比如最近60个交易日），只需选取对应的行范围即可。

## 五、总结

- **核心方法**：基于日对数收益率计算皮尔逊相关系数
- **必备数据**：个股与上证指数的收盘价序列（日期对齐）
- **关键步骤**：计算收益率 → 删除缺失值 → 调用CORREL或corr函数
- **动态视角**：建议同时计算滚动相关性，观察是否随时间变化

这样计算出的相关系数，可以用于判断个股是否“跟随大盘”、适合进行指数对冲、或者作为马丁格尔策略中选股的一个参考维度（相关性越低，分散效果越好）。
