# BOLL三轨简化输出格式说明

## 📋 优化内容

将BOLL三轨价差信息从多行显示优化为**单行紧凑格式**，提高可读性并节省空间。

## 🎯 输出格式对比

### ❌ 优化前（多行显示）
```
📊 BOLL上轨价差: -5.27% (下方)
📊 BOLL中轨价差: +4.40% (上方) ← 此轨最近
📊 BOLL下轨价差: +16.27% (上方)
```

### ✅ 优化后（单行显示）
```
📊 BOLL上轨-5.27% | 中轨+4.40% ← 此轨最近 | 下轨+16.27%
```

## 💡 格式说明

**统一格式：** `BOLL上轨{价差}% | 中轨{价差}% ← 此轨最近 | 下轨{价差}%`

**关键要素：**
1. **正负号**：直观表示价格相对位置
   - `+` 正值：价格在轨道上方
   - `-` 负值：价格在轨道下方

2. **分隔符**：使用 `|` 分隔三个轨道信息

3. **最近轨道标识**：在最接近的轨道后面标注 `← 此轨最近`

### 核心代码

```python
# 计算各轨道价差的绝对值
up_abs = abs(signal.boll_up_diff_pct)
middle_abs = abs(signal.boll_middle_diff_pct)
down_abs = abs(signal.boll_down_diff_pct)

# 找出最近的轨道
min_diff = min(up_abs, middle_abs, down_abs)
if min_diff == up_abs:
    closest_track = "上轨"
elif min_diff == middle_abs:
    closest_track = "中轨"
else:
    closest_track = "下轨"

# 构建简化的BOLL信息显示
up_marker = " ← 此轨最近" if closest_track == "上轨" else ""
middle_marker = " ← 此轨最近" if closest_track == "中轨" else ""
down_marker = " ← 此轨最近" if closest_track == "下轨" else ""

boll_info = f"BOLL上轨{signal.boll_up_diff_pct:+.2f}%{up_marker} | 中轨{signal.boll_middle_diff_pct:+.2f}%{middle_marker} | 下轨{signal.boll_down_diff_pct:+.2f}%{down_marker}"
logger.success(f"📊 {boll_info}")
```

## ✅ 优势分析

### 1. **空间效率**
- 从3行减少到1行，节省67%的垂直空间
- 在飞书通知中更加紧凑，避免消息过长

### 2. **可读性提升**
- 一眼可以看到所有三个轨道的信息
- 使用分隔符 `|` 清晰区分各部分
- 最近轨道标识醒目

### 3. **信息完整性**
- 保留了所有关键信息：价差百分比、正负号、最近轨道标识
- 移除了冗余的"上方/下方"文字（正负号已足够表达）

### 4. **一致性**
- 控制台日志和飞书通知使用相同格式
- 便于快速扫描和对比
