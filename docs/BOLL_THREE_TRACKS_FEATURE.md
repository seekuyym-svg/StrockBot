# BOLL三轨完整价差显示功能说明

## 📋 功能概述

优化后的BOLL布林带分析功能会同时显示价格与**上轨、中轨、下轨**的完整价差信息，并自动标识出**最接近的轨道**。

## 🎯 核心改进

### 1. **统一的计算基准**
所有价差计算均使用**信号价格**作为分母（符合BOLL指标价差计算基准规范）：

- **上轨价差**: `(BOLL上轨 - 价格) / 价格 × 100%`
- **中轨价差**: `(BOLL中轨 - 价格) / 价格 × 100%`
- **下轨价差**: `(价格 - BOLL下轨) / 价格 × 100%`

### 2. **正负号含义**
- **正值 (+)**: 价格在轨道**上方**
- **负值 (-)**: 价格在轨道**下方**

### 3. **最近轨道标识**
系统会自动计算三个价差的绝对值，找出最小值对应的轨道，并在该行末尾标注 `← 此轨最近`。

## 💡 技术分析价值

### 1. **快速判断价格位置**
通过正负号和"上方/下方"标识，一眼看出价格相对各轨道的位置关系。

### 2. **识别关键支撑/阻力**
- **最近轨道**往往是当前的关键支撑或阻力位
- 如果最近的是**下轨**，说明价格接近支撑位，可能是买入机会
- 如果最近的是**上轨**，说明价格接近阻力位，可能是卖出时机

### 3. **判断趋势强度**
- 价格远离中轨、靠近上轨 → 强势上涨
- 价格远离中轨、靠近下轨 → 弱势下跌
- 价格靠近中轨 → 震荡整理

### 4. **预警突破信号**
- 当价差绝对值 < 1% 时，价格即将突破该轨道
- 结合信号类型可以提前预判交易机会

### 计算逻辑

```python
# 统一使用信号价格作为分母
if price > 0:
    # 上轨价差：(上轨 - 价格) / 价格 * 100%
    boll_up_diff_pct = (market_data.boll_up - price) / price * 100
    
    # 中轨价差：(中轨 - 价格) / 价格 * 100%
    boll_middle_diff_pct = (market_data.boll_middle - price) / price * 100
    
    # 下轨价差：(价格 - 下轨) / 价格 * 100%
    boll_down_diff_pct = (price - market_data.boll_down) / price * 100

# 找出最近的轨道
up_abs = abs(boll_up_diff_pct)
middle_abs = abs(boll_middle_diff_pct)
down_abs = abs(boll_down_diff_pct)
min_diff = min(up_abs, middle_abs, down_abs)

if min_diff == up_abs:
    closest_track = "上轨"
elif min_diff == middle_abs:
    closest_track = "中轨"
else:
    closest_track = "下轨"
```
