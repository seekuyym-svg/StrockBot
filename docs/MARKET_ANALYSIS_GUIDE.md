# 市场研判功能说明

## 📋 功能概述

系统现已支持基于**RSI指标**和**BOLL布林带**的综合市场研判功能，自动分析市场状态并给出操作建议。

## 🎯 研判规则

### 规则1: 回调信号 ⚠️

**条件：**
- RSI > 70（超买区）
- 价格距离BOLL上轨 ≤ 5%

**输出：**
```
💡 研判: 回调 - RSI超买且价格接近BOLL上轨，市场可能过热，存在回调风险 ⚠️
```

**含义：** 市场处于超买状态且价格接近压力位，短期内可能出现回调。

**操作建议：**
- 谨慎追高，避免在高位建仓
- 已有持仓可考虑分批止盈
- 等待回调后的低吸机会

---

### 规则2: 反弹机会 ✅

**条件：**
- RSI < 30（超卖区）
- 价格距离BOLL下轨 ≤ 5%

**输出：**
```
💡 研判: 反弹 - RSI超卖且价格接近BOLL下轨，市场可能超跌，存在反弹机会 ✅
```

**含义：** 市场处于超卖状态且价格接近支撑位，短期内可能出现反弹。

**操作建议：**
- 关注反弹机会，可能是较好的建仓时机
- 已有持仓可继续持有等待反弹
- 注意控制仓位，避免过度加仓

---

### 规则3: 震荡行情 🔄

**条件：**
- 30 ≤ RSI ≤ 70（中性区）
- 价格距离BOLL中轨 ≤ 5%

**输出：**
```
💡 研判: 震荡 - RSI中性且价格在BOLL中轨附近，市场处于震荡整理阶段 🔄
```

**含义：** 市场处于平衡状态，多空力量相当，价格在中轨附近震荡。

**操作建议：**
- 按照策略正常执行买卖信号
- 适合区间操作，高抛低吸
- 结合其他指标寻找突破方向

---

### 规则4: 暂无明确信号 ⏸️

**条件：** 不满足上述任一规则

**输出：**
```
💡 研判: 暂无
```

**含义：** 当前市场状态不明确，或数据不完整。

**操作建议：**
- 继续观望，等待更明确的信号
- 检查数据源是否正常
- 结合其他技术指标综合判断

## 🔧 技术实现

### 核心模块

**文件位置:** `src/utils/market_analyzer.py`

**主要类和方法:**
```python
class MarketAnalyzer:
    """市场分析器"""
    
    # 配置参数
    RSI_OVERBOUGHT = 70        # 超买阈值
    RSI_OVERSOLD = 30          # 超卖阈值
    BOLL_CLOSE_THRESHOLD = 5.0 # 接近轨道的阈值(%)
    
    @staticmethod
    def analyze_market_condition(rsi, boll_up, boll_middle, boll_down) -> str:
        """分析市场状态"""
        
    @staticmethod
    def get_analysis_description(result) -> str:
        """获取详细说明"""
```

**便捷函数:**
```python
from src.utils.market_analyzer import analyze_market, get_analysis_description

# 使用示例
result = analyze_market(rsi, boll_up, boll_middle, boll_down)
desc = get_analysis_description(result)
```

### 集成位置

1. **调度器** (`src/utils/scheduler.py`)
   - `_log_important_signal()` - 重要信号输出
   - `_log_wait_signal()` - 等待信号输出

2. **飞书通知** (`src/utils/notification.py`)
   - `_build_message()` - 构建飞书消息体

## ⚙️ 参数调整

如需调整研判阈值，编辑 `src/utils/market_analyzer.py`：

```python
class MarketAnalyzer:
    # RSI阈值配置
    RSI_OVERBOUGHT = 70  # 超买阈值（可调）
    RSI_OVERSOLD = 30    # 超卖阈值（可调）
    
    # BOLL轨道接近阈值（百分比）
    BOLL_CLOSE_THRESHOLD = 5.0  # 距离轨道5%以内视为"接近"（可调）
```

**建议值：**
- **保守策略**: `BOLL_CLOSE_THRESHOLD = 3.0`（更严格）
- **正常策略**: `BOLL_CLOSE_THRESHOLD = 5.0`（默认）
- **激进策略**: `BOLL_CLOSE_THRESHOLD = 8.0`（更宽松）

## 💡 使用建议

### 1. 结合多个指标

研判结果应作为**辅助参考**，而非唯一决策依据：
- ✅ 结合马丁格尔策略信号
- ✅ 观察成交量变化
- ✅ 关注大盘整体趋势
- ✅ 考虑市场情绪和消息面

### 2. 不同场景的应用

**回调信号出现时：**
- 如果是BUY信号，谨慎建仓或减少仓位
- 如果已有持仓，考虑部分止盈
- 等待回调后的更好买点

**反弹信号出现时：**
- 如果是WAIT状态，可能是建仓良机
- 如果已加仓，可继续持有等待反弹
- 设置好止损位

**震荡信号出现时：**
- 按策略正常执行
- 适合做T+0短线操作
- 关注突破方向

### 3. 风险提示

⚠️ **重要提醒：**
- 研判结果基于技术指标，存在滞后性
- 单一指标可能有误判，需综合判断
- 极端行情下技术指标可能失效
- 始终做好风险控制，设置止损

---

## 📝 注意事项

1. **数据完整性**: 需要RSI和BOLL三轨数据完整才能进行研判
2. **阈值选择**: 默认5%的"接近"阈值适用于大多数情况，可根据标的特性调整
3. **边界处理**: RSI=70不算超买，RSI=30不算超卖（严格大于/小于）
4. **日志级别**: 
   - 有明确研判结果：INFO级别
   - 结果为"暂无"：DEBUG级别（避免刷屏）

## 🔄 与其他功能的协同

### 与马丁格尔策略配合

- **回调信号 + BUY信号**: 谨慎建仓，减少初始仓位
- **反弹信号 + WAIT状态**: 积极建仓，把握反弹机会
- **震荡信号 + ADD信号**: 正常加仓，区间操作

### 与飞书通知配合

- 实时接收研判结果
- 快速判断市场状态
- 及时调整交易策略

### 与信号持久化配合

- 历史研判记录可回溯分析
- 统计各类研判的准确率
- 优化研判参数配置
