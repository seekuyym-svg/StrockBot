# RSI指标功能说明

## 📋 功能概述

系统现已支持在信号输出时显示**RSI（相对强弱指数）**指标，并自动判断市场状态。

## 🎯 RSI指标说明

RSI（Relative Strength Index）是衡量价格变动速度和幅度的技术分析指标，取值范围0-100。

### RSI区域判断标准

| RSI值范围 | 区域 | 标识 | 含义 |
|-----------|------|------|------|
| RSI > 70 | 超买区 | 🔴 ⚠️ | 市场可能过热，存在回调风险 |
| 30 ≤ RSI ≤ 70 | 中性区 | 🟡 | 市场处于正常波动范围 |
| RSI < 30 | 超卖区 | 🟢 ✅ | 市场可能超跌，存在反弹机会 |

## 💡 交易建议

### 超买区（RSI > 70）
- ⚠️ **风险提示**: 市场可能过热
- 💡 **操作建议**: 
  - 谨慎追高，避免在高位建仓
  - 已有持仓可考虑分批止盈
  - 关注可能的回调机会

### 中性区（30 ≤ RSI ≤ 70）
- ✅ **正常状态**: 市场处于合理波动范围
- 💡 **操作建议**: 
  - 按照策略正常执行买卖信号
  - 结合其他指标综合判断

### 超卖区（RSI < 30）
- ✅ **机会提示**: 市场可能超跌
- 💡 **操作建议**: 
  - 关注反弹机会，可能是较好的建仓时机
  - 已有持仓可继续持有等待反弹
  - 注意控制仓位，避免过度加仓

## 🔧 技术实现

### 数据模型

**Signal模型** (`src/models/models.py`):
```python
class Signal(BaseModel):
    # ... 其他字段 ...
    
    # RSI指标
    rsi: Optional[float] = None  # RSI值
```

**MarketData模型** (`src/models/models.py`):
```python
class MarketData(BaseModel):
    # ... 其他字段 ...
    
    rsi: Optional[float] = None  # RSI指标
```

### 策略引擎

在生成所有类型的信号时（BUY/ADD/SELL/STOP/WAIT），都会从`market_data.rsi`获取RSI值并传递到Signal对象中。

### 调度器输出

在`src/utils/scheduler.py`中的以下方法添加了RSI输出：
- `_log_important_signal()`: 重要信号输出
- `_log_wait_signal()`: 等待信号输出

## 📱 飞书通知支持

系统会将RSI指标同步到飞书通知中，格式与控制台输出保持一致。

### 飞书通知示例

**买入信号通知:**
```
🟢 买入信号

标的: 港股创新药ETF广发 (sh.513120)
信号类型: BUY
当前价格: ¥1.281
涨跌幅: +0.71%
目标份额: 14,000 份
平均成本: ¥1.281
BOLL上轨-5.56% | 中轨+4.22% ← 此轨最近 | 下轨+13.99%
📈 RSI: 74.32 (🔴 超买区 ⚠️)

原因: 初始建仓：买入14000份，成本1.281元/份
```

**加仓信号通知:**
```
🔵 加仓信号

标的: 港股创新药ETF广发 (sh.513120)
信号类型: ADD
当前价格: ¥1.250
涨跌幅: -2.42%
目标份额: 10,000 份
平均成本: ¥1.265
BOLL上轨-7.85% | 中轨+2.15% ← 此轨最近 | 下轨+12.15%
📈 RSI: 25.80 (🟢 超卖区 ✅)

原因: 加仓10000份，成本1.250元/份，累计1次加仓
```

### 配置方法

在 `config.yaml` 中配置飞书通知：

```yaml
notification:
  feishu:
    enabled: true
    webhook_url: "https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx"
    notify_signals: ["BUY", "SELL", "ADD", "STOP"]
```

**注意**: 
- WAIT信号默认不发送飞书通知，避免信息过载
- 可在 `notify_signals` 中添加 "WAIT" 来启用观望信号通知

### 测试飞书通知

运行以下命令测试飞书通知（包含RSI信息）：

```python
from src.utils.notification import get_feishu_notifier

notifier = get_feishu_notifier()
notifier.test_notification()
```

## 📝 注意事项

1. **RSI值为None时**: 不显示RSI信息（数据源可能未提供）
2. **精度**: RSI值保留两位小数
3. **实时性**: RSI基于最新的市场数据计算
4. **参考性**: RSI仅作为辅助参考指标，应结合其他技术指标和策略信号综合判断

## 🔄 与其他指标的协同

RSI与BOLL布林带指标配合使用效果更佳：

- **RSI超买 + 价格接近BOLL上轨**: 强烈回调信号
- **RSI超卖 + 价格接近BOLL下轨**: 强烈反弹信号
- **RSI中性 + 价格在BOLL中轨附近**: 震荡行情

建议在决策时综合考虑多个技术指标，提高交易胜率。
